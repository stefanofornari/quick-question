# Quick Question User Stories Index

This index lists the user stories derived from `uzz/specs/scope.md`.

## Domains

### Component
Reusable JavaFX component for embedding a webview and selecting LLM chat endpoints.

- [US-000001](component/US-000001) — Embed a webview-based LLM launcher
- [US-000002](component/US-000002) — Select an LLM from a configured list
- [US-000003](component/US-000003) — Load LLM entries from configuration
- [US-000006](component/US-000006) — Provide predefined LLM providers
- [US-000007](component/US-000007) — Persist WebView session cookies across restarts

### Demo
Demo JavaFX application used to showcase the component and provide simple configuration UI.

- [US-000004](demo/US-000004) — Provide a demo application with simple configuration UI
- [US-000005](demo/US-000005) — Dynamically add WebChat entries to the dropdown
- [US-000008](demo/US-000008) — Show predefined providers with Google-login warnings
