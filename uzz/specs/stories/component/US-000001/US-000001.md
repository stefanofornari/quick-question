# User Stories for Domain: Component

## [US-000001] Embed a webview-based LLM launcher
**As a** JavaFX application developer,
**I want** to embed a reusable component that opens a webview to an LLM chat page,
**So that** I can provide quick access to an LLM inside my application.

### Technical Context & Constraints
- **Target System:** JavaFX desktop applications
- **Toolchain / Rules:** JavaFX, Maven, JUnit 5, TestFX, AssertJ, headless testing, TDD workflow, snake_case test names, use `then()` for BDD assertions

### Acceptance Criteria
```gherkin
Scenario: Component opens with a default LLM page
  Given the component is created with a valid default LLM URL
  When the component is displayed
  Then the webview loads the default LLM page

Scenario: Component handles an invalid default URL
  Given the component is created with an invalid default LLM URL
  When the component is displayed
  Then the component shows a navigational failure state without crashing
```
