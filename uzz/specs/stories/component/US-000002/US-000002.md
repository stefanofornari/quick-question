# User Stories for Domain: Component

## [US-000002] Select an LLM from a configured list
**As a** JavaFX application user,
**I want** to choose an LLM from a list shown above the webview,
**So that** I can switch between supported LLM chat pages quickly.

### Technical Context & Constraints
- **Target System:** JavaFX desktop applications
- **Toolchain / Rules:** JavaFX, Maven, JUnit 5, TestFX, AssertJ, headless testing, TDD workflow, snake_case test names, use `then()` for BDD assertions

### Acceptance Criteria
```gherkin
Scenario: User selects a configured LLM
  Given the component is displaying a list of configured LLMs
  When the user selects one LLM from the list
  Then the webview navigates to the selected LLM URL

Scenario: No LLMs are configured
  Given the component is created with an empty LLM configuration
  When the component is displayed
  Then the selector shows no entries and the webview remains on its current page
```
