# User Stories for Domain: Component

## [US-000006] Provide predefined LLM providers with Google-login warnings
**As a** Developer,
**I want** a predefined list of providers to give to the component
**So that** I do not have to search and updates the most common providers, their
            urls and their capabilities.

### Technical Context & Constraints

- Predefined providers:
  | Provider | URL | googleLogin |
  |---|---|---|
  | Anthropic Claude | https://claude.ai | false |
  | ChatGPT | https://chatgpt.com | true |
  | Perplexity | https://www.perplexity.ai | true |
- The component itself does not show predefined providers in the choice list; it
  instead provides a Property with the predefined providers so that a developer
  The developer will then be able to use it like this:
  ```java
  private WebChatViewController webChatViewControllerController;
  ...
  webChatViewControllerController.entriesProperty().set(new DefaultProviders());
  ```

### Acceptance Criteria
```gherkin
Scenario: DefaultProviders
  Given the class DefaultProviders
  When the class is istantiated
  Then the instance contains all providers above

```
