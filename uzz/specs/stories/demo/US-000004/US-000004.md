# User Stories for Domain: Demo

## [US-000004] Provide a demo application with simple configuration UI
**As a** project evaluator or developer,
**I want** a demo JavaFX application with a simple configuration UI,
**So that** I can quickly verify how the component behaves and how to supply LLM entries.

### Technical Context & Constraints
- **Target System:** JavaFX desktop application
- **Toolchain / Rules:** JavaFX, CommonsFX, AtlantaFX, Maven, JUnit 5, TestFX, AssertJ, headless testing, TDD workflow, snake_case test names, use `then()` for BDD assertions

### Acceptance Criteria
```gherkin
Scenario: Demo application starts with configuration controls
  Given the demo application is launched
  When the main window is displayed
  Then the user can see the component and a simple configuration UI

Scenario: Demo application updates the component configuration
  Given the user edits the LLM configuration in the demo UI
  When the user applies the changes
  Then the component reflects the updated LLM list
```
