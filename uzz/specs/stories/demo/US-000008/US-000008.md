# User Stories for Domain: Demo

## [US-000008] Show predefined providers with Google-login warnings
**As a** user,
**I want** the demo application to populate the selector with the predefined providers and warn me when a provider does not support Google login,
**So that** I can quickly choose a common LLM provider and know to use email login when necessary.

### Technical Context & Constraints
- **Target System:** JavaFX desktop application
- **Toolchain / Rules:** JavaFX, CommonsFX, AtlantaFX, Maven, JUnit 5, TestFX, AssertJ, headless testing, TDD workflow, snake_case test names, use `then()` for BDD assertions

### Acceptance Criteria
```gherkin
Scenario: Demo shows predefined providers in the selector
  Given the demo application is launched
  When the main window is displayed
  Then the selector contains the predefined provider names

Scenario: Selecting a predefined provider navigates to its chat URL
  Given the demo application is launched
  When the user selects "ChatGPT" from the selector
  Then the component navigates to "https://chatgpt.com"

Scenario: Warning appears for providers with unsupported Google login
  Given the demo application is launched
  When the user selects "Anthropic Claude" from the selector
  Then a warning is shown next to the selector stating Google login is not supported
  And the warning advises the user to use email login

Scenario: No warning appears for providers with supported Google login
  Given the demo application is launched
  When the user selects "Perplexity" from the selector
  Then no Google-login warning is shown
```
